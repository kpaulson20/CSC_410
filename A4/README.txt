**********************ABOUT**********************
Converting the matrix multiplication program
and the trapezoidal rule program to parallel
using the OpenMP technique. Then, compare the
runtimes for both programs to thier sequential
version. Used #pragma omp parallel for collapse()
for the matrix multiplication and #pragma omp 
parallel, #pragma omp for, and #pragma omp 
critical for the trapezoidal program.
