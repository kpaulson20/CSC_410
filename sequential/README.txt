matrixMul: multiplies two pre-filled matrices
A and B and multiplies them together, then
stores the results in an unused matrix C

arraySum: takes in all the elements in the
array and adds them together and prints 
out the total of all numbers

Nqueens: uses backtracking to search for 
all valid locations for which a a queen can
be placed on an 8 x 8 chessboard
